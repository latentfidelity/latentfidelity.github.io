Our latest Research and Products

Synthetic Characters

A real-time video agent API that lets you build fully custom conversational characters. These video agents can have any appearance and any visual style, with full control over voice, personality, knowledge and actions.

Built on a recently launched world model, Characters lets teams generate expressive digital personas from a single image, with zero fine-tuning required.

Learn more

Introducing World Model Alpha

A general world model built to simulate reality in real time. Interactive, controllable and general-purpose.

The model comes in three variants: interactive worlds for explorable environments, avatars for conversational characters, and robotics simulation for manipulation.
